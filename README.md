## 📖 Overview

Elevora is a premium digital marketing agency website built to showcase services, attract clients, generate leads, and establish a strong online presence.

The website features a modern UI/UX design, smooth animations, responsive layouts, lead generation forms, portfolio showcases, testimonials, and a complete admin management system.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/elevora.git
```

### Navigate Into Project

```bash
cd elevora
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Website will run on:

```bash
http://localhost:3000
```

---

## 🔐 Environment Variables

Create a `.env` file:

```env
MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_jwt_secret

NEXT_PUBLIC_SITE_URL=http://localhost:3000

EMAIL_USER=your_email

EMAIL_PASSWORD=your_password
```
